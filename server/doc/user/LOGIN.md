# 登录

## V1版本

### 请求方式
| 前缀 | 版本 | 请求方式 | 链接 |
| :---: | :---: | :---: | :---:|
| api | v1 | post | /happy_pig/api/v1/user/login |

<br/>

### 请求参数

---------------------------------
|  key  |   类型   | key是否必填 | 备注 |
| :---: | :------: | :--------: | :---:|
| email | String | 是 | 邮箱 |
| password | String | 是 | 密码 |

<br/>

### 返回结果

----------------------------
|  key  |   类型   |  备注 |
| :---: | :------: | :---:|
| code | int | 1001:成功、1002:失败、10005:邮箱或密码错误、10004:邮箱未注册|
| msg | string | 提示语 |
| data | object | 返回的数据 |

<br/>

#### 返回成功事例

-------------------------
```json
{
  "code": 1001,
  "data": {
  },
  "msg": "成功"
}
```
